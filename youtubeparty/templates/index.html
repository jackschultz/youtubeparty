Collaboratively Share Youtube Playlists

Designed for sharing music. Create a name for a room, give others the code, upload video links, and listen to them play.

How it works

Create a room

Simply type in a name for the room, such as a location or theme, and press create. You're automatically redirected to your new room.

Share the code

Each room comes with a six digit code to share. Give it to friends to collaboratively create a playlist everyone is happy with. Or share it with the world and see what bizzare videos show up in the feed.

Select videos

Useing the builtin search functionality, search for the video you want to queue right from the room. Click the upload button and watch as it's put in the playlist.
